> 总体观感：
> 阅读这本书的契机完全得益于微信读书的神秘网友们，他们三番两次地在我阅读的另一本理财入门的新手指南书里提到这本《认知觉醒》。一开始我不以为然，但当它出现在我眼前的次数越来越多，我无法不对它好奇。
>
> 全书共六章，阅读了三分之二，我决定对此先做出我个人的判断，再去仔细查阅其他书友的锐评。
>
> 首先，这本书不适合直接用打分制来评价。它有目标受众，对于初出茅庐的高中大学生，初入社会的年轻人，或者浑浑噩噩百无聊赖的普通中年人（一般来说，忙碌的人会有自己追求的核心锚点，不太需要此类书籍的指导），值得一看，看了总比没看好。
>
> 而对于心理学/脑科学/神经科学等等专业知识熟知者，或者通过各种前置条件已经有核心锚点的高度自我觉知者，我想这本书就像有点难以下咽的鸡汤，有很多过时或者不专业不严谨的论据。
>
> 这本书大约有80%的内容是铺垫和举例论证，有营养的核心观点在我个人观感里不超过20%。我是一个有意识，对阅读质量有追求的刻薄读者，因此我希望不要在这本鸡汤型书籍上不要浪费时间。
>
> 在此，我更倾向于把一条浑身布满鱼刺的骨架挑出来，剩下最值得一尝的那部分鱼肉。

​

# 第一章 大脑

这一章关于“三重大脑”模型的论述非常不严谨：

```
本能脑（爬行动物，3.6 亿年）→ 情绪脑（哺乳动物，2 亿年）→ 理智脑（人类新皮层，250 万年）
```

这个框架已经过时，作为神经科学事实已被否定。

**这段话中正确的部分**：

· 爬行动物确实依赖快速、刻板的反射行为（如战斗/逃跑、捕食、求偶），这些行为由脑干和基底节等深层结构调控。

· 哺乳动物在这些基础结构之上，演化出了更复杂的情感与社会行为调节系统。

· 人类保留着这些深层结构，且它们在唤醒、习惯形成、自动反应中起作用。

**这段话中被简化和误导的部分**：

· “最初的‘本能脑’”不成立：爬行动物的脑并非“只有”反射模块。研究表明，爬行动物表现出复杂的学习能力、空间记忆甚至社会认知（如蜥蜴的领地识别、鳄鱼的亲代抚育），这些需要超越“简单反射模块”的神经基础。

· “结构很简单”是误解：爬行动物的脑干和基底节在微观层面极其复杂，涉及多核团、多递质系统的精细调控，并非“简单”。

· “让爬行动物对环境快速做出本能反应”忽视了灵活性：爬行动物的行为并非完全固定。它们能根据环境线索调整捕食策略、在熟悉环境中走捷径，说明存在一定的认知灵活性。

· “遇到心仪的异性就追求”将复杂行为归因于单一脑区：求偶行为涉及感觉输入、动机评估、运动输出的全脑协作，仅归因于“爬行动物脑”是过度简化。



## 本章值得记忆点：

严格上第一章来说没有必须要看的，重要点在后续章节其实会展开说。

```

```

# 第二章 潜意识

模糊是困扰之源，分三类：

**认知模糊**：多读书、多建立概念；

**情绪模糊**：当你感到心里有说不清、道不明的难受的感觉时，赶紧坐下来，向自己提问。

**行动模糊**：选择节点上给明确指令，即：量化（30min、一篇500字）、有步骤、可验收（设置DDL）

## 本章值得记忆点：

```
有启发的内容，就触发熔断点，立刻停止读书。停止读书之后做什么呢？围绕这个触发点对自己提问。
·为什么刚才这个点让我有启发？
·我能够把这个启发点用在3个不同的事情上吗？
·这个启发点有没有其他类似的知识？
　
　模糊，正是人生困扰之源。而人生也像是一场消除模糊的比赛，谁的模糊越严重，谁就越混沌；谁的模糊越轻微，谁就越清醒。
　
　总之，人生就是一场消除模糊的比赛，我们比拼的不仅仅是成长的速度，还有成长的模式。在这条赛道上，领先的群体都有意无意地做着同一件事：消除认知、情绪和行动上的模糊。
```

# 第三章 元认知

元认知：

对自己思考过程的思考，也就是监控自己大脑。

元认知包含三点

1. 自我觉察：知道自己懂什么、不会什么。
2. 自我调控：方法不对就调整。
3. 自我评价：做完判断自己做得好不好。

> 监控自己的注意力，然后将其集中到自己需要关注的地方。

元认知能力强的一个突出表现是：

对模糊零容忍。换句话说，就是想尽一切办法让自己找出那个最重要的、唯一的选项，让自己在某一个时间段里只有一条路可以走。

自控力：

> 做选择是一件极为耗能的事情，如果没有与之匹配的清醒和定力，绝大多数人最终都会被强大的天性支配，去选择娱乐消遣。在有约束的环境下我们反而效率更高，生活更充实。

自控的本质不是压制本能，而是**在"元时间"（选择节点）上停 0.x 秒**，让理智介入。

```
当下觉知→审视第一反应→明确主张；全天日程清晰；长远目标想清楚意义。
```

## 本章值得记忆点：

我们每天都要做那些让自己感到有些困难但又可以通过努力来完成的事情，即跳出舒适区，避开困难区，处在拉伸区。

```
二十多岁的圈子本来就很局限，身边都是同年龄、同眼界、同层次的人，大家纠结的烦恼、认知的边界都差不多，很难有人能拉高我的眼界、给我更高维的思考。

现实里想结识一个通透、有格局、有深度认知的高人太难了，还要磨合圈子、人情、距离。但读书不一样，一本好书就是一位顶级前辈在掏心掏肺把一辈子的阅历、思维、避坑经验全教给我。

不用迎合、不用社交、不用看人脸色，只要翻开书，就能跨越时间和阶层，和古今中外的厉害人物对话，借他们的脑子想问题，借他们的阅历走人生。

这真的是年轻人最便宜、最高效的升维方式。

读书不是为了立刻变现，而是悄悄拓宽眼界、拉高认知、稳住内心，让我们在平凡的生活里，拥有一个只属于自己、辽阔又自由的精神天地。
```

# 第四章 专注力

走神："做 A 想 B"。

根源：当下太无聊→求有趣，或当下太痛苦→求舒适。

```
老和尚说：“得道前，砍柴时惦记着挑水，挑水时惦记着做饭；得道后，砍柴即砍柴，担水即担水，做饭即做饭。”
```

解法：**把感受收回行动本身**（跑步就感受抬腿迎风，吃饭就感受嚼劲），砍柴即砍柴。

## 本章值得记忆点：

```
　　《施瓦辛格健身全书》:你的肌肉只会在被强迫超负荷运作之后才会生长。它们不会对任何不足的负荷作出反应，哪怕只差了一点点。
```



# 第五章 学习力

原文就不摘抄了，又臭又长。

打卡的三大弊端：

1.**动机转移**（从学习本身移到"完成任务"）；

2.**认知闭合**（一打完卡动机归零）；

3.**任务心态破坏身心合一**。

改良：**用记录代替打卡**、**设下限不设上限**（每天背 5 个单词比 20 个更易持续）。

**无反馈不学习**

总结就是：

> 主动学习者要懂得：先做一个垃圾出来，再反馈-打磨至最佳状态。
>
> 作品→反馈→头部→跃迁

## 本章值得记忆点：

```
在舒适区边缘游走：
跑步/英语/阅读/健身皆如此。距离太远的事物（突然的大把自由、暴富）人也掌控不住。

深度学习：
学习层级：听书 < 自己读 < 读+摘抄 < 读+思维导图 < 读+践行+输出教授（缝接）。
三步：高质量输入 → 深度缝接 → 输出教授。
浅学习只关注输入，深学习关注输出。

主动休息：
"番茄工作法"思路：【25】分钟极度专注 + 5 分钟主动休息

其中极度专注的时间往往根据个人状态随时调整，比如进入状态也可以改为【120min】极度专注，【30min】主动休息
```

# 第六章 行动力

成长悖论：

想先看到结果再行动 → 永远看不到结果。

行动为先，不计成本与结果地去行动。这个世界有时也会奖励那些不计得失的"傻瓜"。

**懂得百点不如改变一点**。把"改变"作为最高标尺。

## 本章值得记忆点：

```
清晰力才是行动力。
早上第一件事决定"增强回路"方向（刷手机→负回路，读书/运动→正回路）。
把目标写下来（待办→计划→实际→备注）。
```

# 第七章 情绪力

**心智带宽：唯有富足，方能解忧**

五帖觉知药：环境觉知、目标觉知（少即是多）、欲望觉知、情绪觉知、闲余觉知（适当设限）

**游戏心态：幸福的人，总是在做另外一件事**

自我决定理论：关系/能力/自主。

**"我并不是在做这件事，我只是在做另外一件事"**：

跑步测试 = 追逐游戏

写作业 = 挑战速度。

> **动机转移**：为自己而做 + 为玩而做。

# 第八章 早冥读写跑

**第一节 早起**：放弃闹钟、感知睡眠周期节点（1.5h 倍数）、循序渐进、中午补午休。早起多出"第三个 8 小时"。

**第二节 冥想**： brains 里 7±2 个"小球"，冥想练的是让 7 个球协同。物理上增大大脑皮层表面积+灰质厚度，且治情绪熵（轻球加码、重球正视）。

**第三节 阅读**：顶级思想的最低成本获取方式。选书（需求匹配）> 读书量。读书最终目的是**改变**。

**第四节 写作**：费曼技巧核心——**用简单的话 + 自己的语言**讲清楚，最好让外行听懂。类比是认知桥梁。

**第五节 运动**：运动让大脑长新神经元，但**新神经元需要 28 天在"运动后脑力活动"中存活**。"运动+学习"才是完整闭环。复杂运动（瑜伽/舞蹈）比纯有氧更促突触。
